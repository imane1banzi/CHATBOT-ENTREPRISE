Créer un ticket Jira pour Mediametrie – problème mot de passe expiré :

1. Ouvre Jira et connecte-toi avec ton compte.
2. Clique sur "Créer" pour générer un nouveau ticket.
3. Dans la section **Type de ticket**, choisis "Demande".
4. Sélectionne le **workspace Mediametrie**.
5. Dans le champ **Résumé**, indique le problème : "Mot de passe expiré – accès Mediametrie".
6. Dans la **Description**, détaille la situation : date, ID utilisateur, etc.
7. Affecte le ticket à **toi-même** ou à la personne responsable.
8. Vérifie toutes les informations et clique sur **Créer / Submit**.
9. Note le numéro de suivi du ticket pour référence future.
